# Tauri + React + Typescript

This template should help get you started developing with Tauri, React and Typescript in Vite.

## 运行方式

### 开发模式

```bash
pnpm tauri dev
```

这会同时启动 Vite 开发服务器和 Tauri 桌面窗口。

### 其他命令

- `pnpm dev` - 只启动前端 Vite 开发服务器（浏览器预览）
- `pnpm build` - 构建生产版本
- `pnpm tauri build` - 构建桌面应用安装包

### 前提条件

- 已安装 Node.js 和 pnpm
- 已安装 Rust（Tauri 后端需要）
- macOS 需要安装 Xcode Command Line Tools
